# Yeni Konu Ekleme Şablonu

```text
[ANA_KLASÖR] altında [SIRA]-[KONU_ADI] adlı bir öğrenme konusu oluştur.

Beklentiler:
- Türkçe README yaz.
- Konuyu gösteren küçük, çalıştırılabilir bir Python örneği ekle.
- Haricî bağımlılıklar varsa requirements.txt oluştur.
- Ağ erişimi veya büyük model indirme gerekiyorsa çalıştırma yerine kod iskeleti sağla ve README'de açıkla.
- Mevcut klasörleri veya ilgisiz dosyaları değiştirme.
- Uygun bir doğrulama komutu çalıştır ve sonucu özetle.
```
