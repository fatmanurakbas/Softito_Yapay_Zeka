# Ajan Çalışma Alanı

Bu klasör, `Softito_Yapay_Zeka` eğitim deposunda içerik üreten veya gözden geçiren yapay zekâ ajanları için ortak çalışma notlarını barındırır. Amaç; yeni konu eklerken klasör düzenini, dil tonunu ve doğrulama yaklaşımını tutarlı tutmaktır.

## Başlangıç noktası

1. Önce [project-map.md](project-map.md) dosyasından ilgili öğrenme alanını bulun.
2. [coding-guidelines.md](coding-guidelines.md) kurallarını okuyun.
3. Yapılacak işe göre `prompts/` altındaki uygun şablonu kullanın.
4. Yalnızca istenen klasörde değişiklik yapın; ilgisiz dosyaları düzenlemeyin.

## Temel ilkeler

- Açıklamalar ve README dosyaları Türkçe yazılır.
- Her konu, mümkünse küçük ve çalıştırılabilir bir örnek içerir.
- Haricî servis, model indirme veya API anahtarı gerektiren örnekler açıkça belirtilir.
- Örnek veri, çıktı ve gizli bilgiler sürüm kontrolüne bilinçsizce eklenmez.
- Değişiklikten sonra ilgili kod veya en azından sözdizimi kontrol edilir.

Bu dosyalar ajanların çalışma kılavuzudur; uygulama kaynak kodunun parçası değildir.
