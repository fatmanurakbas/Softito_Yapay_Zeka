# 04 — Kök Bulma ve Lemmatization

Türkçe eklemeli bir dil olduğundan kök bulma, İngilizceye göre daha dikkatli ele alınmalıdır. Bu klasördeki yöntem eğitim amaçlı, kurala dayalı ve sınırlı bir stemmer'dır; gerçek lemmatization için dilbilgisel çözümleme gerekir.

```bash
python stemming_lemmatization.py
```

Çıktıyı mutlaka örnekleyin: agresif ek kesme, `kitaplar` gibi yararlı biçimleri bozabilir. Üretim için Zemberek gibi Türkçe odaklı bir çözüm değerlendirilebilir.
