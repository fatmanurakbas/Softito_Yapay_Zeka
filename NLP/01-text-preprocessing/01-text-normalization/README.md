# 01 — Metin Normalleştirme

Normalleştirme, aynı anlama gelen farklı yazımları tutarlı hale getirir ve model için gürültü oluşturan parçaları temizler.

Bu örnek; Unicode biçimlendirme yapar, metni küçük harfe indirger, URL/e-posta/para tutarlarını yer tutucularla değiştirir ve gereksiz boşlukları temizler. Yer tutucular bilgiyi silmek yerine türünü korur.

```bash
python normalization.py
```

Göreve göre sayıların veya emojilerin korunup korunmayacağını yeniden değerlendirin.
