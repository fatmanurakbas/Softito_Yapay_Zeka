# GloVe — Global Vectors

GloVe, kelimelerin pencereler içinde birlikte görünme sayılarını kullanır. Amaç, iki kelimenin vektör çarpımının birlikte-görünme sayısının logaritmasını yaklaşık olarak öğrenmesidir.

```bash
python glove_cooccurrence.py
```

Buradaki sade uygulama eğitim mantığını göstermek içindir; üretim için optimize edilmiş GloVe araçları ve çok daha büyük derlemler gerekir.
