# 05 — Metin Sınıflandırma

Bu uygulama, CSV dosyasındaki Türkçe ürün yorumlarını pozitif/negatif olarak sınıflandırmak için Multinomial Naive Bayes kullanır. Kelime sayımı özellikleri, sınıflandırıcının olasılık hesabına girer.

```bash
python naive_bayes_classifier.py
```

Örnek veri çok küçük olduğundan doğruluk gerçek performansı temsil etmez. Çalışmanın amacı veri bölmesi, eğitim, tahmin ve hata analizinin temel akışını göstermektir.
