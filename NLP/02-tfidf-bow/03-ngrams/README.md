# 03 — N-Gramlar

N-gram, ardışık `n` tokenın oluşturduğu birimdir. Bigramlar kelime sırasını kısmen korur ve `çok iyi`, `hiç güzel değil` gibi kısa ifadeleri yakalayabilir.

```bash
python ngrams.py
```

N değeri arttıkça sözlük büyür ve veri seyrekleşir. Başlangıçta unigram + bigram kombinasyonu çoğu metin sınıflandırma problemi için iyi bir deneme noktasıdır.
